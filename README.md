2. 📅 backtest (地狱级回测与模型重训)

定位： 策略的“事后审判”与“AI 记忆更新”大脑。

算力聚焦时间：每周五深夜自动运行。

硬核逻辑 (超越时空的进化)：

记忆遗忘曲线 (Time-Decay Memory)：AI 掌握了“遗忘”。在训练 Random Forest 时，注入了指数时间衰减权重。越古老的交易数据对 AI 决策的干预越低，它将永远活在“当下的市场周期”里。

MAE/MFE 归因分析：不仅测试盈亏，更追踪每一笔盈利单在盘中到底承受了多大的回撤折磨 (Max Adverse Excursion)，打破盈亏黑盒。

AI 影子过滤：通过平行回测，对比“原始胜率”与“经 AI 过滤后胜率”，用数据证明护城河的厚度。
