⚙️ 核心运行模式解析

由于抛弃了高频盘中噪音，本系统将所有算力汇聚于每日与每周最关键的时刻。

智能基因 (The AI Core)：

第一顺位：AI 胜率主导：系统不再依靠人类主观赋分，而是通过模型直接输出 🤖 AI胜率: 68.5% 作为选股的第一顺位排序标准。

2. 📅 backtest (地狱级回测与模型重训)

定位： 策略的“事后审判”与“AI 自我迭代”大脑。

算力聚焦时间：每周五深夜自动运行。

🚨 宏观信用风控触发 (Macro Credit Spread)

原理：跨市场监测高收益垃圾债 (HYG) 与 避险美债 (IEF) 的比值。

动作：一旦发现聪明资金从垃圾债疯狂出逃，系统强制扣减全市场健康分，大幅压缩凯利建仓上限。领先半步逃顶。

动作：检测到价格近高而 RSI 衰竭，直接将总分清零。拒绝在主力诱多时接盘。
